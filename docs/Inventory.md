Elite Build CRM Inventory and Admin Console

The Admin should be able to add new projects to the Inventory from the Admin Console and define every attribute of the project that shows up in the Inventory from the Admin console. The sales team should have access to see the details of the project. 

Some of the default attributes of projects are listed below, however the admin should have access to remove existing fields for a project and add new fields to a project when needed.

The inventory page should be searchable. 
Should list out all the projects that Elite Build has undertaken.
When click on the project name the details of the project should show up.
Should have a filter as well.

The admin console should be an independent console from where:
1. Inventory can be defined.
2. User access control could defined.
